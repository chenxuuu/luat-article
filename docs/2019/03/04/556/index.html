判断SIM卡所属的运营商

SIM卡所属的运营商可通过查询sim卡的ismi前缀来判断

AT命令

使用AT命令AT+CIMI可以得到sim卡的ismi

lua

使用sim.getImsi()接口可以得到sim卡的ismi

前缀

46000, “CHINA MOBILE”, “CN” 中国移动
46001, “CHN-CUGSM”, “CN” 中国联通
46002, “CHINA MOBILE”, “CN” 中国移动

其他前缀可以搜索运营商PLMN对照表来查看
