dtu阿里云设备影子

1，新建产品

这次我们直接新建一个基础版就可以

2，配置dtu

这次订阅主题比较特别不在设备的topic里而是文档说明里

也就是这样

/shadow/update/a1JvMM3l4n0/${deviceName}

/shadow/get/a1JvMM3l4n0/${deviceName}

根据自己的改一下

3，测试

测试影子json

  },

}
