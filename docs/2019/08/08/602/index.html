一、服务器端创建产品

         如果没有账户，注册一个新账户

二、设备端代码修改，生成升级包

            例如：

         设备端本地烧录的旧固件版本信息为：

         core：Luat_V0019_ASR1802

         main.lua中PROJECT：LUAT_IOT_SERVER_UPDATE

         main.lua中VERSION：1.0.0

         模块型号：Air720D

 

         远程升级的新固件版本信息为：

         core：Luat_V0021_ASR1802

         main.lua中PROJECT：LUAT_IOT_SERVER_UPDATE

         main.lua中VERSION：1.0.1

         模块型号：Air720D
