阿里云影子文件操作指南

不废话,直接上代码吧

流程

  1. 开机
  2. 连接阿里云
  3. 定义shadow更新的topic
  4. 发送一个{“method”:”get”}到/shadow/update的topic,以获取最新的shadow version
  5. 阿里云下发最新的shadow,读取verison,更新本地状态
  6. 根据业务逻辑发送最新shadow或者处理下发的shadow状态
