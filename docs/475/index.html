为服务器安装phpMyAdmin方便管理mysql服务器

在Linux操作系统的服务器下来操作和使用mysql对于菜鸟来说，比较费劲，因为操作mysql数据库需要各种各样的指令，我们更希望能够有一种直观的可视化的操作数据库的方法，那么我们可以在服务器上安装phpMyAdmin。

phpMyAdmin可以让我们可以通过图形界面的方式来操作mysql数据库，非常好用。接下来我来与大家分享一下在服务器上安装phpMyAdmin的方法。


我的服务器操作系统为64位的Centos6.5，安装的mysql版本为5.5，关于mysql5.5的安装方法，可以参考帖子：为Linux服务器安装mysql数据库教程


接下来分享安装phpMyAdmin的方法：


1、使用putty软件远程登录我的服务器，在服务器上下载phpMyAdmin.tar.gz压缩包。

我要把phpMyAdmin.tar.gz压缩包下载到服务器的download目录下，那么我执行 cd /web/eoair/download 命令进入download目录。

然后，百度一下phpMyAdmin的官网，进去找一下phpMyAdmin.tar.gz的下载链接，如下图所示：

我们先把上图中红框那个下载选项下载到本地电脑上，然后使用ftp软件，将下载下来的压缩文件上传到我服务器的/web/eoair/download目录下。

然后执行 ll 命令来查看一下我服务器 /web/eoair/download目录下的文件，如下图所示：

上图中红框圈中的文件就是phpMyAdmin的压缩文件包。

0 x  U3 s' [, ~

2、解压phpMyAdmin的压缩文件包

首先执行 cd /web/eoair/download 进入到download目录，然后 执行 ll 来查看该目录下的文件如下图所示：

接下来执行命令 tar -zxvf phpMyAdmin-4.5.2-all-languages.tar.gz 来将该文件解压，解压结束后，执行 ll 查看文件，如下图所示：

上图中红框圈起来的内容就是解压之后的 phpMyAdmin目录，文件名有点长，我给改一下：

执行 mv phpMyAdmin-4.5.2-all-languages  phpMyAdmin 将原来那么长的名字改为 phpMyAdmin，修改后执行 ll 查看文件如下所示：

上图中红框圈起来的为改名后的 phpMyAdmin。

& V- j- b( `8 g/ w# |/ [

3、将步骤2中的 phpMyAdmin目录移动到 /web/eoair/www目录下

执行 mv phpMyAdmin  /web/eoair/www  将/web/eoair/download目录下的phpMyAdmin移动到了服务器  /web/eoair/www路径下，那么我们执行 cd /web/eoair/www 进入该目录，然后执行 ll 查看文件如下图所示：

上图中红框圈起来的就是我们移动过来的 phpMyAdmin文件。

7 {' Z4 B8 k4 g- l

4、接下来，打开浏览器，输入我的域名/phpMyAdmin后，显示界面如下图所示：

这说明phpMyAdmin可以使用了，接下来，输入用户名和密码，点击“执行”，然后就登录进去了。接下来，就可以操作mysql数据库了。
