网卡模式下获取Air720的运行状态

背景

  1. 需要获取联网状态,设备识别号(IMEI),ICCID,重启设备
  2. 通过AT可以获取网络状态,SIM的ICCID
  3. linux下需要修改内核驱动才能出AT口

突破口

  1. 设备自带一个网页管理工具
  2. 该工具需要登录,基于变种的Digest realm登录验证
  3. 网页是通过ajax访问/xml_action.cgi读取和操作设备
  4. 必须先完成一次登录,才能通过普通的Digest realm机制访问xml_action
