天马Air720G模块客户问题分析报告

客户名称：天马

模块：Air720UG

问题现象：量产测试阶段，测量模块VBAT和GND短路

模块数量：线上测试有20块，天马寄来5块进行分析，有4块VBAT和GND短路，1块测量是没有短路的，但是不能工作，电压被拉低到1.17V，用万用表二极管档测量模块的正负极都有0.34V左右的压降

问题分析：测量VBAT管脚与GND短路，模块PMU供电端VBAT TVS卸下，再次测量短路现象消失

原因分析：由于静电击穿（ESD),电压尖峰等原因，VBAT端TVS被击穿，导致TVS失效
