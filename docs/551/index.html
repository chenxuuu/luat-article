Air720系列-死机dump数据功能使用说明

一、下载安装SecureCRT

可通过百度云下载SecureCRT软件，并按照文件包中的提示进行安装注册。

链接: https://pan.baidu.com/s/13lhILGSYB4Vilxqb9mo8Ng

提取码: a3kt

二、模块死机时发送AT*EXASSERT=1会让模块不重启，此时设备管理器就出现一个新的端口号，如图所示：

三、打开SecureCRT软件选择Quick connect选项或者使用快捷键Alt+Q来配置连接数据

四、配置链接文件将protocol设置为serial，port设置为设置为第二步中的新的端口号，baud rate设置为115200，点击connet。

五、连接成功后选择Transfer选项中的Receive Ymodem...选项来dump数据。

六、dump数据时间较长，期间不要操作SecureCRT软件

由于工具原因导致无法dump  Filesystem.bin文件，可以使用ctrl+c退出。

七、dump出来的文件可以在Options选项中的Session Options看到下载的位置
