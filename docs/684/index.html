做出快速简单的带有语音播放天气的定时闹钟（使用air202和阿里云物联网平台）

首先：

准备air202开发板、杜邦线、喇叭

开通阿里云物联网平台

打开连接阿里云物联网平台的luat程序（大佬们之前写过：air202luat二次开发设备接入阿里云https://doc.luatos.wiki/523/）


其次：

1.登陆阿里云，点击API市场

2.选择墨迹天气免费版气象服务

3.查看API接口,记录调用地址、请求方式


4.然后,点击立即购买

5.购买完成后,点击进入墨迹天气

6.查看产品详情，点击cityid对照表，点击token查询方式，将token下载到桌面


7.查找所在的城市，记录id号

8.点击调试工具,将刚才记录的数据填写进去

    点击发送请求

    查看当前天气数据格式

    记录APPCODE

9.在iot studio中的新建服务开发

    将所用到的节点搭建起来

    将刚才记录的数据填写进去

    配置各节点

10.用 python脚本解析墨迹天气的数据

11.修改air202的程序


最后：

12.点击HTTP节点的调试

    air202开始语音播放天气
13.配合阿里云web可视化开发，就可以快速简单的做出带有播放天气的定时闹钟了！
